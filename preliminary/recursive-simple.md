### Problem: Calculate Factorial

Create a program that calculates the factorial of a given integer.

#### Input:

* An integer `<span>n</span>`(0 ≤ n ≤ 20)

#### Output:

* The factorial of `<span>n</span>`

### Sample Input and Output:

#### Example 1:

**Input:5**

Output::120


#### Example 2:

Input:0

Output:1
