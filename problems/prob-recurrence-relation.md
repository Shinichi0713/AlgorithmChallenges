**Problem:**A sequence is defined as follows:

* a_1 = 2
* a_n = a_{n-1} × 3 + 1 for n≥2

Your task is to find the value of ana_n for a given nn.

**Input Format:**
A_n integer n (1≤n≤20).

**Output Format:**
The value of ana_n in the sequence.

**Example Input 1:**

```
4
```

**Example Output 1:**

```
82

```
